🌐 Currency Converter Web App | Projeto Final UNIPÊ
Este é o projeto final da disciplina de Desenvolvimento Web do curso de Ciência da Computação (UNIPÊ). Trata-se de uma aplicação web moderna para conversão de moedas estrangeiras com taxas atualizadas em tempo real.

🎯 O Projeto
A aplicação foi construída para demonstrar o domínio de tecnologias web essenciais, focando na integração entre interface de usuário e serviços de dados externos (APIs).

🔑 Principais Desafios Técnicos:
Consumo de API Fetch/Axios: Requisições assíncronas para obter dados de câmbio.

Manipulação Dinâmica do DOM: Atualização dos valores na tela sem necessidade de recarregar a página.

Design Responsivo: Interface adaptável para dispositivos móveis e desktops (Mobile First).

Tratamento de Erros: Gestão de estados para falhas de conexão ou moedas não encontradas.

🛠️ Tecnologias e Ferramentas
Estrutura: HTML5 / Semântica Web.

Estilização: CSS3 (Flexbox, Grid) / [Ex: Bootstrap ou Tailwind].

Lógica: JavaScript (ES6+) / [Ex: React ou Vue, se usou algum].

API de Dados: [Ex: AwesomeAPI / ExchangeRate-API].
